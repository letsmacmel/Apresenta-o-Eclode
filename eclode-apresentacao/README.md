# Eclode — apresentação viva

Aplicação web local para criar uma apresentação reativa e multipágina.

## Abrir

Execute um servidor local dentro desta pasta:

```powershell
$env:PORT=4174
node server.js
```

Depois abra `http://localhost:4174`.

## Recursos

- textos editáveis diretamente nos slides;
- salvamento automático no navegador;
- criação, duplicação e exclusão de páginas;
- quatro layouts;
- cor e intensidade reativa por página;
- navegação por teclado;
- modo de apresentação em tela cheia;
- exportação e importação de backup JSON.
